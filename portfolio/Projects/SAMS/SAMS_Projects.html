Typing Coach

This was our first basic game using JAVAscript, HTML and a little CSS. Within the JAVAscript file you had an array with a list of 20 words. Every 3 seconds a random word would be selected and displayed on the page at a random location. At the top of the page you had a form where you could type the words that were displayed on the page. By using an if staemtnt we where able to cross refernce wether the typed value matched any of the values diplayed on the page and if it was true the word was removed. The objective of the game was to constantly keep the page clear of any words.
